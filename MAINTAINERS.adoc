== Review policy

* At least one maintainer review is required before merge to `main`.
* Changes to the Idris proof routes (`routes/`) and the demos (`examples/`)
  must type-check under Idris 2 (`idris2 --check`) before review — proofs are
  load-bearing here, so a red type-check blocks merge.
* Documentation-only changes (`.adoc`, `.well-known/`, AI manifests) follow the
  lighter Perimeter-3 path in `CONTRIBUTING.md`.
