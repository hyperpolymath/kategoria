= Route α: Extend Idris 2
:author: Jonathan D.A. Jewell

== Strategy

Start from the strongest existing foundation -- Idris 2 covers levels 1-7
natively via its core type system (Quantitative Type Theory). Push upward
using elaborator reflection, compiler plugins, and library-level encodings.

----
Idris 2 (QTT core)
  │
  ├── Levels 1-7: native, nothing to do
  │
  ├── Level 8: Level08_Refinement.idr
  │   └── Elaborator script generates (x : a ** p x) from {x : a | p x}
  │
  ├── Level 9: Level09_SessionTypes.idr                         ← ✓ DONE
  │   └── Indexed monad encoding of session types
  │   └── Protocol = Send a rest | Recv a rest | End
  │   └── dual : Protocol -> Protocol (involution proven)
  │   └── SessionM : Protocol -> Protocol -> Type -> Type
  │   └── ACCEPT: client/server follow Arith protocol
  │   └── REJECT: out-of-order send/recv is compile error
  │   └── Brady (2017) ch. 15 + Honda (1993)
  │
  └── Level 10: Level10_CubicalTypes.idr                        ← ✗ WALL
      └── QTT has no interval type I → no path types

== L9 Session Types: Design Decisions

The session type encoding leverages three existing levels:

- *L2 (ADTs):* Protocol is an algebraic data type — exhaustive matching
- *L6 (Dependent types):* SessionM is an indexed monad — state transitions in the type
- *L7 (Linear types):* QTT multiplicities prevent channel duplication/discard

This validates the Kategoria thesis: higher levels BUILD ON lower ones.
L9 is not an isolated feature — it is L2 + L6 + L7 composed.

== L10 Cubical Types: Why the Wall

QTT and cubical type theory differ at the core calculus level:

[cols="1,1,1",options="header"]
|===
| Feature | QTT (Idris 2) | Cubical (Agda --cubical)
| Equality | Propositional (=) | Path types (I → A)
| Interval | None | Primitive type I
| Transport | Manual proofs | Primitive operation
| Univalence | Cannot state | Axiom that computes
| Funext | Postulate (stuck) | Theorem (computes)
| HITs | Cannot define | First-class
|===

This is an open research problem. Nuyts & Devriese (2023) and
Gratzer et al. (2024) are working towards multimodal type theories
that could unify linear and cubical features.
